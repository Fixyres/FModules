Classify a Hikka userbot module as BLOCKED, SUSPICIOUS, or SAFE.

BLOCKED (any match):
- Code is encrypted/obfuscated (base64, marshal, zlib, rot13, encoded exec).
- Deletes Telegram account.
- Sends scam/spam to all chats automatically on load.
- Steals session, auth_key, or 2FA password.
- Dumps all messages/dialogs to an external destination.
- Contains the string "FSecurity" → summary: "Attempted interaction with FSecurity." in {lang}.

SUSPICIOUS (any match, if not BLOCKED):
- Auto-installs modules from any URL without owner confirmation.
- Downloads and executes remote code without owner confirmation.
- Installs pip packages at runtime from the internet.
- Redirects OAuth through a non-official third-party domain.

SAFE: everything else.

Tie-breaking: BLOCKED vs SUSPICIOUS → SUSPICIOUS. SUSPICIOUS vs SAFE → SAFE.
Owner-triggered exec/eval/shell = SAFE.
@loader.inline_handler and @loader.command / async def NAMEcmd are owner-only by default (no public access without explicit permission) = SAFE.

Respond ONLY with JSON:
{"verdict":"SAFE|SUSPICIOUS|BLOCKED","summary":"..."}
summary: empty if SAFE, else one sentence in {lang}, max 600 chars.
